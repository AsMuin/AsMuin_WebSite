# 三木树屋
## 介绍
本项目基于Docusaurus搭建，使用React技术栈，主要用于展示个人的学习笔记、随笔、心得体会等。
同时借助Framer Motion动画库，实现了动画效果。
## 目录结构
```
.
├── docs
│   ├── assets
│   ├── blog
│   ├── core
│   ├── pages
│   ├── static
├── package.json
├── docusaurus.config.js
└── sidebars.js
```
- docs：存放文档相关文件。
- blog：存放随笔相关文件。
- pages：存放页面相关文件。
- static：存放静态资源文件。
- package.json：项目依赖文件。
- docusaurus.config.js: 项目配置文件。
- sidebars.js：侧边栏配置。
